## 结论
这个实验目前只对比了4类9个模型，从对比结果来看，高斯过程的总体误差最小。在偏性-方差分解方面，高斯过程的偏性和方差也都是最小的。排名第二的集成方法-随机森林，比高斯过程略差一些。这也从侧面反映了高斯过程在低维空间中的有效性。未来我们将看到更多"回归模型的结果，便于把握各个模型的特点。
